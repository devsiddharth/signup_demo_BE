# 🔐 User Registration & Authentication System

A full-stack user authentication application built using **Spring Boot**, **React**, and **MySQL**. The project demonstrates user registration, secure password encryption, authentication using Spring Security, and JWT-based authorization.

---

## 🚀 Features

- User Registration
- Secure Password Encryption (BCrypt)
- User Login
- Spring Security Integration
- JWT Authentication
- Role-based Authorization (Upcoming)
- React Frontend with Axios
- MySQL Database Integration
- RESTful APIs

---

## 🛠️ Tech Stack

### Backend
- Java 17

- Spring Data JPA

- Maven
- MySQL

### Frontend
- React
- TypeScript
